# 背景
wealth-wallet-<<llm model name>>> 是一个包含后端和前端的应用，目标是
```
做一个财富钱包应用，包括前端和后端，功能包括记录某个账户下的存款和负债，账户可能是银行账户或者券商账户或者交通卡钱包，券商账户可以记录股票持仓，货币可能是人民币或者美元或者港元，默认是人民币，钱包应用能自动计算当前所有账户的余额（存款减去负债），显示折线图对于某一个账户，其他基本功能也应该包含，前端用react技术，后端用Python,包含一个饼状图来显示各个账户占比，前台背景加一些antd的图标或样式，更加时尚
```
分别基于 claude, deepseek, glm, kimi, minimax, qwen, auto 大模型开发了对应的代码
